# Vision Index

The vision layer holds the highest-level creative direction for the novel and the prose-style rules that govern how it is written. Load the narrative brief first for any task; load the style files only when drafting or revising prose.

Read first: [narrative-brief.md](narrative-brief.md). It is short and meant to be read before anything else.

| File | Authority | Status | Load when |
| --- | --- | --- | --- |
| [narrative-brief.md](narrative-brief.md) | vision | active-canon | Starting any task; it frames premise, promise, world, characters, themes, tone, and the canon priority order. |
| [style/index.md](style/index.md) | style-rule | active-support | Drafting or revising prose; it indexes the eleven independently loadable style files (core prose, viewpoint, dialogue, character voices, AI dialogue, technology in prose, emotion and moral content, pacing, formatting, prohibited patterns, revision checklist). |

## Common tasks

- To orient on the whole project, read [narrative-brief.md](narrative-brief.md).
- To draft a chapter in the right voice, open [style/index.md](style/index.md) and load only the style files the chapter needs.
- For where canon authority lives by subject, see [../00-governance/canon-hierarchy.md](../00-governance/canon-hierarchy.md).
