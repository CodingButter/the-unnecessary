# Continuity Index

> No manuscript chapters have been approved. These files contain pre-draft starting conditions only.

This directory holds the continuity baseline for The Unnecessary: what is true about the
story world and its cast at Book One time zero (the morning of October 3, 2053), derived
strictly from the approved canon under `docs/20-canon/`. Because no manuscript chapters
are approved, this is a pre-draft baseline. It records starting conditions only. It does
not record any planned, future, or yet-to-be-drafted event as established continuity.
Planted setups may be named, but their payoffs are marked PLANNED and not-yet-delivered,
never as occurred. When chapters are later drafted and approved, continuity is tracked
forward from this known baseline rather than reconstructed.

## Read first

Start with [global-continuity.md](global-continuity.md). It establishes the world state at
time zero (year, social structure, uneven infrastructure decline, enclave status, Greater
Detroit location baselines, and what is functional versus failed) that every other file
assumes as background.

## The seven continuity dimensions

This baseline records the seven minimum continuity dimensions named in the master spec:

1. Global starting conditions
2. Character starting states
3. Character knowledge states
4. Relationship starting states
5. Technology starting states
6. Unresolved threads
7. Setups and payoffs

## Contents

| Dimension | Entry point | What it records |
|---|---|---|
| Global starting conditions | [global-continuity.md](global-continuity.md) | The world state at time zero: year, social structure, infrastructure decline level, enclave status, Greater Detroit location baselines, what is functional versus failed. |
| Character starting states | [character-states/index.md](character-states/index.md) | One file per established character or intelligence: location, condition, situation, goals, and stated secrets at time zero. |
| Character knowledge states | [knowledge-state/index.md](knowledge-state/index.md) | One file per character: what is known, falsely believed or privately held, and not yet known, kept separate to prevent reveal leakage. |
| Relationship starting states | [relationships/index.md](relationships/index.md) | Per-pairing relationship baselines: alliance, tension, debt, trust level, and last known interaction. |
| Technology starting states | [technology-state/index.md](technology-state/index.md) | Operational state of key systems: Crown, Morrow, and the infrastructure layer, marking what is up, degraded, or failed. |
| Unresolved threads | [unresolved-threads.md](unresolved-threads.md) | Open questions and dangling threads at story start, each framed as an unresolved question whose answer is not yet delivered. |
| Setups and payoffs | [setups-and-payoffs.md](setups-and-payoffs.md) | Planted setups paired with intended payoffs, every payoff labeled PLANNED and not yet delivered. |
