# Knowledge State Index

> No manuscript chapters have been approved. These files contain pre-draft starting conditions only.

character or intelligence knows, what they falsely believe or privately hold, and what
they do not yet know. The not-yet-known column is kept strictly separate so no later
reveal leaks into a character's starting knowledge. All entries derive from approved
canon, chiefly the character-knowledge timeline and the secret timeline.
