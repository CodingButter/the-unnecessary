## Decision 041: Chapter Blueprints Are Created One Chapter at a Time

**Status:** Locked for Current Workflow
**Category:** Planning

### Decision

The Plot Outline provides the complete high-level book structure.

Detailed Chapter Blueprints are created close to the time each chapter is drafted.

### Previous or Alternative Direction

Creating complete scene-by-scene blueprints for all 36 chapters before drafting was rejected as the default process.

### Reason

Prose drafting will reveal better character reactions, pacing, and continuity details.

Creating all blueprints too early would produce large amounts of planning that might need repeated revision.

### Consequences

Recommended rhythm:

1. Create Chapter Blueprint.
2. Review it.
3. Draft the chapter.
4. Revise the chapter.
5. Approve the chapter.
6. Update continuity.
7. Create the next blueprint.

A preliminary blueprint for the following chapter may be maintained.

### Affected Documents

- Novel Development and Canon Guide
- Chapter Blueprint workflow
- Project Status file

### Reconsider Only If

The drafting process proves more effective with a larger planning buffer.
