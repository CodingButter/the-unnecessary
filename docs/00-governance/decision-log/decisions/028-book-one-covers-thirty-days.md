## Decision 028: Book One Covers Thirty Days

**Status:** Locked for Current Draft
**Category:** Plot structure

### Decision

Book One runs from October 3 through November 1, 2053.

Morrow exists for less than one month during the first novel.

### Reason

A compressed timeline creates urgency while keeping Morrow’s rapid development tied to existing models, hardware, and knowledge rather than impossible learning from nothing.

### Consequences

- Morrow’s capabilities must emerge from integration and architecture.
- Character travel and recovery must fit the calendar.
- Relationship changes need to feel intense but plausible.
- Technical work still requires realistic physical time.

### Affected Documents

- Master Timeline
- Plot Outline
- Chapter Blueprints

### Reconsider Only If

Chapter development proves the events require more time. The change must be made before substantial manuscript drafting.
