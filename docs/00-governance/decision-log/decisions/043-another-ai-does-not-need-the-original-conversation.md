## Decision 043: Another AI Does Not Need the Original Conversation

**Status:** Locked for Current Workflow
**Category:** AI handoff

### Decision

Another capable language model should be able to continue the project using the active documents without access to the original development conversation.

### Reason

The project’s core decisions, canon, workflow, and rejected directions are being externalized into documents.

### Requirements

The new AI should receive:

- Novel Development and Canon Guide
- Narrative Brief
- Story Bible
- Character Bible
- World and Technology Rules
- Master Timeline
- Plot Outline and Chapter Map
- Chapter Blueprint Template
- This Decision Log

Once drafting begins, it should also receive:

- Current Continuity Ledger
- Previous approved chapter
- Previous Chapter Blueprint
- Current Style Guide

### Consequences

The project should not depend on conversational memory.

Any important new decision made in conversation must be added to the relevant document.

### Affected Documents

- Novel Development and Canon Guide
- README
- Project Status file
- AI handoff prompts

### Reconsider Only If

A new collaborator lacks sufficient context capacity. In that case, focused document excerpts should be prepared.
