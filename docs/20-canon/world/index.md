# World Canon Index

This index maps the world-canon files for the novel and points to the authority that owns each topic. It only summarizes and links. It never restates canon. For any load-bearing fact, open the file named here and read it directly.

## Purpose of this directory

The `world/` directory holds the canonical shape of the setting for The Unnecessary: the core premise and time period, how society is stratified, how ordinary infrastructure erodes, where the wealthy shelter, what Mars and Aurelia are, how government and corporate power relate, the themes and central questions, the multi-book trajectory, the Book One arc, and the rules that keep the world consistent. Specific places live in the [locations subtree](./locations/index.md).

## Read first

Start with [core-premise.md](./core-premise.md). It establishes the working title, format, genre, tone, core premise, time period, and the final promise of the story. Everything else in this directory builds on that framing.

## Authority and status

Every file in this directory carries `authority: world-canon` and `status: active-canon`, and all draw from the `Story Bible.md` monolith. They are authoritative on world facts and outrank both mem0 memory and the knowledge graph in any conflict. This index carries `status: active-support` and establishes nothing on its own. Some topics are deliberately split across domains: government-and-corporations defers to the character and technology domains for Adrian Kade and Crown, mars-and-aurelia defers to the technology domain for Mars engineering and to the timeline for Aurelia milestones, and consistency-rules defers to the technology domain for hard plot restrictions and Morrow behavior.
