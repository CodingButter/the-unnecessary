# Artificial Intelligence

## Purpose of this directory

This directory holds the technology canon for artificial intelligence in **The Unnecessary**: how AI capability is tiered in 2053, how the story treats machine consciousness, and the technology authorities for the two named superintelligences, Crown and Morrow. It governs what AI can and cannot do as a matter of technology. The behavioral identities of Crown and Morrow as characters live in the character canon, not here.

This index summarizes and links. It does not restate canon. For any load-bearing fact, open the file named below.

## Read first

Start with [intelligence-levels.md](intelligence-levels.md). It classifies the four tiers of AI and is the frame the rest of this directory assumes. After that, [crown.md](crown.md) and [morrow.md](morrow.md) describe the two named superintelligences, and [crown-vs-morrow.md](crown-vs-morrow.md) compares them.
