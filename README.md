# The Unnecessary

A near-future, grounded-cyberpunk novel project. This repository holds the
development materials for the book, not a software product: there is no
application to build or run here, only canon, planning, manuscript, continuity,
and governance documents.

## Start here

This README is a signpost, not an instruction set. Two destinations carry the
real guidance.

- **`CLAUDE.md`** (repo root) is the per-session entry point. Read it first.
  It states what the project is, where authority lives, and how to use
  persistent memory before you do any story work.
- **`docs/00-governance/`** holds governance and navigation: the canon
  hierarchy, the context-loading guide, the directory indexes, and the rules
  for handling conflicts. Go there to learn how to find and load only the files
  a given task needs.
  - `docs/00-governance/index.md` (pending: created in a later phase) will be
    the governance directory's index and the recommended first read inside it.

## Status

The repository is mid-migration toward a structured documentation tree. Work
happens on the `migration` branch and integrates to `main`. Folder names and
locations may still shift during the migration; defer to `CLAUDE.md` and the
governance documents for the current authoritative layout.
